# webDependencyFetcherAPI
Get list of dependencies for a url resource (e.g. if html -> list css, js, img)

```bash
# brew install uv
./app.py
curl --request GET --url 'http://localhost:8000' --data-urlencode 'urls=https://www.bbc.co.uk/'
```
